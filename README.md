# Automated-Data-Pipeline-Health-Monitoring-TLS-Certificate-Remediation
## Description:

Built an end-to-end automated system using Matillion, Fivetran, and Snowflake to detect, validate, and auto-fix broken data pipelines caused by daily server reboots. The solution eliminated manual intervention by automatically identifying failed connectors, validating TLS/SSL certificates, triggering recovery syncs, and sending audit-based alerts—reducing resolution time from hours to minutes and ensuring seamless morning operations.
